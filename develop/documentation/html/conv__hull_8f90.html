CONV_HULL computes the upper envelope of the convex hull of a set.

Author
Thomas R. Cameron, Davidson College
Nikolas I. Steckley, Portland State University
Date
2017
Purpose:
 CONV_HULL uses the monotone chain algorithm to compute the convex hull.
Parameters
[in]a
Double precision array of dimension n, contains the log of the moduli of polynomial coefficients,
 ordered from constant to leading.
[in]n
Integer, size of a.
[in,out]h
Integer array that contains the indexes corresponding to the upper envelope of the convex hull.
[in,out]c
Integer, number of indeces in the upper envelop of the convex hull.
