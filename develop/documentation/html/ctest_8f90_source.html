7 ! (IR,A(IR)) at the vertex (I,A(I)), up to within the tolerance *
8 ! TOLER. If convexity holds then the function is set to .TRUE., *
9 ! otherwise CTEST=.FALSE. The parameter TOLER is set to 0.4 by default. *
10 !************************************************************************
11 ! Input variables: *
12 ! A : vector of double *

14 !************************************************************************
15 ! Output: *
16 ! .TRUE. if the angle formed by (IL,A(IL)), (I,A(I)), (IR,A(IR)) at *
17 ! the vertex (I,A(I)), is convex up to within the tolerance *
18 ! TOLER, i.e., if *
19 ! (A(I)-A(IL))*(IR-I)-(A(IR)-A(I))*(I-IL)>TOLER. *
20 ! .FALSE., otherwise. *
21 !************************************************************************
36 !************************************************************************
