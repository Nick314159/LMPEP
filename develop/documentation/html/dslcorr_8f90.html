DSLCORR computes the Lagueree correction term of real polynomial with real root approximation, while `‘dividing out’' previously computed roots.

Author
Thomas R. Cameron, Davidson College
Nikolas I. Steckley, Portland State University
Date
2017
Purpose:
 DSLCORR calculates y1=p'(t)/p(t) and y2=-(p'(t)/p(t))', where t=er(ind) is the current eigenvalue approximation,
 and then updates y1 and y2 by subtracting the appropriate sum of preivously computed roots indexed by 1,...ind-1.
 From there the Laguerre correction term can be computed as deg/(y1+-sqrt((deg-1)*(deg*y2-y1**2)),
 where +- is choosen to maximize the denominator.
Parameters
[in]p
Double precision array of dimension (deg+1), contains polynomial coefficients, ordered from constant to leading.
[in]alpha
Double precision array of dimension (deg+1), contains moduli of polynomial coefficients.
[in]tol
Double precision, used to determine potential convergence of eigenvalue approximation..
[in]deg
Integer, degree of the polynomial.
[in]ind
Integer, index of current eigenvalue approximation.
[out]check
Logical, returns false if convergence is reached.
[in,out]er
Double precision array of dimension deg, real part of eigenvalue approximations.
[in,out]ei
Double precision array of dimension deg, imaginary part of eigenvalue approximations.
[out]berr
Double precision number, backward error in current eigenvalue approximation.
Note
MEMORY: O(deg), FLOPS: O(deg)
