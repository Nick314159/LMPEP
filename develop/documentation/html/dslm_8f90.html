DSLM computes the roots of a real polynomial.

Author
Thomas R. Cameron, Davidson College
Nikolas I. Steckley, Portland State University
Date
2017
Purpose:
 DSLM computes the roots of a real polynomial using Laguerre's method.
Parameters
[in]p
Double precision array of dimension (deg+1), contains polynomial coefficients, ordered from constant to leading.
[in]deg
Integer, degree of the polynomial.
[out]er
Double precision array of dimension deg, real part of eigenvalue approximations.
[out]ei
Double precision array of dimension deg, imaginary part of eigenvalue approximations.
[out]berr
Double precision array of dimension deg, backward error of each eigenvalue approximation.
Note
MEMORY: O(deg), FLOPS: O(deg^2)
