DSSTART computes initial estimates to the roots of a polynomial.

Author
Thomas R. Cameron, Davidson College
Nikolas I. Steckley, Portland State University
Date
2017
Purpose:
 DSSTART uses the Newton Polygon of a polynomial to compute initial estimates to that polynomials roots.
Parameters
[in]alpha
Double precision array of dimension (deg+1), contains moduli of polynomial coefficients,
 ordered from constant to leading.
[in]deg
Integer, degree of the polynomial.
[out]er
Double precision array of dimension deg, real part of eigenvalue approximations.
[out]ei
Double precision array of dimension deg, imaginary part of eigenvalue approximations.
Note
MEMORY: O(deg), FLOPS: O(deg)
