LMPEP Documentation

This package is a collection of Fortran 90 subroutines for accurately and efficiently solving the polynomial eigenvalue problem using Laguerre's method.

Authors

  • Thomas R. Cameron, Davidson College, NC
  • Nikolas I. Steckley, Portland State University, OR

In Progress

Currently we are updating the code in the develop section, in the hopes of developing more robust algorithms and better documentation. Also, we are working on the parallelization of these methods.
