ZREVSEVAL evaluates the reversal of a complex scalar polynomial at a complex number.

Author
Thomas R. Cameron, Davidson College
Nikolas I. Steckley, Portland State University
Date
2017
Purpose:
 ZREVSEVAL calculates revp(t), revp'(t), or revp''(t), where revp(t)=t^(deg)p(t^(-1)) and t is a nonzero complex number. What derivative is taken is determined by the parameter der and the computation is done via Horner's method.
Parameters
[in]p
Double complex array of dimension (deg+1), contains polynomial coefficients, ordered from constant to leading.
[in]t
Double complex, number to be evaluated.
[in]deg
Integer, degree of the polynomial.
[in]der
Integer, derivative to be taken (0,1,2).
[out]a
Double complex, return value.
Note
MEMORY: O(deg), FLOPS: O(deg)
