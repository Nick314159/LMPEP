ZSLCORR computes the Lagueree correction term of complex polynomial with complex root approximation, while ``dividing out'' previously computed roots.

Author
Thomas R. Cameron, Davidson College
Nikolas I. Steckley, Portland State University
Date
2017
Purpose:
 ZSLCORR calculates y1=p'(t)/p(t) and y2=-(p'(t)/p(t))', where t=DCMPLX(er(ind),ei(ind)) is the current root approximation, and then updates y1 and y2 by subtracting the appropriate sum of preivously computed roots indexed by 1,...ind-1. From there the Laguerre correction term can be computed as deg/(y1+-sqrt((deg-1)*(deg*y2-y1**2)), where +- is choosen to maximize the denominator.
Parameters
[in]p
Double complex array of dimension (deg+1), contains polynomial coefficients, ordered from constant to leading.
[in]alpha
Double complex array of dimension (deg+1), contains moduli of polynomial coefficients.
[in]tol
Double precision, used to determine potential convergence of eigenvalue approximation..
[in,out]conv
Logical, returns true if convergence is reached.
[in]deg
Integer, degree of the polynomial.
[in]ind
Integer, index of current eigenvalue approximation.
[in,out]er
Double precision array of dimension deg, real part of eigenvalue approximations.
[in,out]ei
Double precision array of dimension deg, imaginary part of eigenvalue approximations.
[in,out]berr
Double precision number, backward error in current eigenvalue approximation.
Note
MEMORY: O(deg), FLOPS: O(deg)
